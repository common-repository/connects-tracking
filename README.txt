== Description ==

# About
This plugin works by appending the Connects Profiling and Conversion Tracking TAGs to your Woocommerce Shop. The Use of this plugin will not change the appearance of your Shop.


== Installation ==


1. Install the plugin
1. Go to settings --> Connects Trackings Settings
1. Insert your Connects ID
1. Activate the Profiling and the Conversion Toggle
